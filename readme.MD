# SD-Action rule repository

This folder contains scripts for replication steps in the article

* **sd\_act\_basic\_flow.py** - the detailed code to show the workflow to mine SD-Action rules, sort and filter the best ones in the article
* **results\_sd\_act\_basic\_flow.txt** - result of the run of the previous code
* **sd\_act\_multiple\_datasets.py** - testing of SD-Action rules on multiple datasets
* **results\_sd\_act\_multiple\_datasets.txt** - result of the run of the previous code
